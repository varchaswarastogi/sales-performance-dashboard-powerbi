📊 Sales Performance Dashboard & 15-Day Forecast (Power BI)
📌 Project Overview

This project is an interactive Sales Performance Dashboard built using Power BI.
It provides insights into sales trends, profit analysis, regional performance, and short-term sales forecasting.
